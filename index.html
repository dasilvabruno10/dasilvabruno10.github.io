Cache Simulator

Developed a Python cache simulator designed to simulate the behavior of a cache memory system given a series of memory addresses. It supports both direct-mapped and set-associative caches with options for configuring block size, number of blocks, associativity, replacement policy, hit time, and miss time. The primary functionality is to compute the hit/miss rate and total access time for a given set of memory accesses.
